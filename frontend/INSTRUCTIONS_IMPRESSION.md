# Instructions d'impression des reçus

## Format recommandé : A5 Paysage

Le format **A5 en orientation paysage** (210 × 148 mm) est le format optimal pour vos reçus.

### Avantages

- **Économique** : Un papier A4 = 2 reçus identiques
- **Professionnel** : Format standard et propre
- **Compatible duplicateur** : Parfait pour papier carbone
- **Pas de déformation** : Découpe précise sans approximation

## Réglages d'impression

### Dans l'application
1. Sélectionnez **Format A5** dans le menu déroulant
2. Cliquez sur **Imprimer**

### Dans la boîte de dialogue d'impression
- **Format papier** : A4
- **Orientation** : Portrait (vertical)
- **Option** : 2 pages par feuille
- **Marges** : Automatiques

### Résultat
Le reçu A5 sera automatiquement dupliqué sur un seul papier A4, prêt à être coupé au milieu.

## Découpe du papier

### Point de coupe
Coupez au centre à **148,5 mm** (mi-hauteur du papier A4)

### Résultat final
Vous obtenez 2 reçus identiques de format A5 (210 × 148 mm)

## Marges pour duplicateur carbone

Les marges sont optimisées pour les duplicateurs :
- **Marge haute** : 10 mm
- **Marge basse** : 10 mm
- **Marges latérales** : 12 mm
- Le texte est éloigné de la ligne centrale pour éviter les problèmes de duplication

## Conseils d'utilisation

1. **Vérifiez l'aperçu** avant d'imprimer
2. **Testez** avec une feuille normale avant d'utiliser le papier duplicateur
3. **Ajustez** les paramètres de votre imprimante si nécessaire
4. **Stockez** les reçus imprimés à plat pour éviter les plis

## Support

Si vous rencontrez des problèmes d'impression, vérifiez :
- Les paramètres de votre imprimante
- La compatibilité du format A5 paysage
- Les marges configurées
- L'option "2 pages par feuille" est bien activée
